# Turm 20 Theatre Website

Modern theatre website built with Next.js and deployed on Render.

## Features
- Event management
- Artist profiles  
- Comment sections
- Responsive design
- Modern performance

## Development
```bash
npm install
npm run dev
```

## Deployment
Connected to GitHub: https://github.com/arminisa95/turm20.git

Deployed on Render (Node.js runtime)

## Content Structure
- `/pages` - Next.js pages
- `/components` - React components
- `/public` - Static assets
- `/styles` - CSS files
